Тестовое задание

Написать автоматизированный тест, который проверяет, правильно ли работает сортировка по цене на сайте Розетка 

Сценарий:

1. Войти на сайт (https://rozetka.com.ua/)
2. Найти какой-нибудь популярный товар (например Iphone 15)
3. Применить сортировку по цене
4. Проверить, что цены на самом деле отсортированы (Цены результатов “нет в наличии” не учитываются)


Использовать подход Page Object или Page Factory.
